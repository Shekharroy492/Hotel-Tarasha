Modern Website Demo

Files:
- index.html — main page with hero, services, portfolio, testimonials, contact
- styles.css — modern responsive styles using CSS Grid/Flexbox and CSS variables
- script.js — vanilla JS for interactivity: dark mode, mobile menu, lightbox, slider, form handling

How to run:
- Open `index.html` in any modern browser (no build step required).

Notes:
- Images use Unsplash public URLs for demo purposes.
- No external JS libraries used — everything is vanilla JS.
- To customize, edit text, replace images, and tweak variables in `styles.css`.
